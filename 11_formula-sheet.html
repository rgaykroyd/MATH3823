11.1 Notation

11.2 Basic rules in probability & statistics

  • Rules of expectation Expectation is linear so for random variables \(X_1\) and \(X_2\), and constants \(a\), \(b\) and \(c\), the \[ \mbox{E}[aX_1+bX_2+c] = a\mbox{E}[X_1]+b\mbox{E}[X_2] +c. \] There is no change to this rule when \(X_1\) and \(X_2\) are independent.

  • Rules of variance By definition, the \(\mbox{Var}[X]\) is given by \[ \mbox{Var}[X] = \mbox{E}[(X-\mbox{E}[X])^2] = \mbox{E}[X]^2-\mbox{E}[X^2] \] with the variance of the sum of two random variables \[ \mbox{Var}[X_1+X_2] = \mbox{Var}[X_1]+\mbox{Var}[X_2]-2\mbox{Cov}[X_1,X_2] \] where \(\mbox{Cov}[X_1,X_2]\) is the covariance which is defined in the following section. If, however, \(X_1\) and \(X_2\) are independent, then the above reduces to \[ \mbox{Var}[X_1+X_2] = \mbox{Var}[X_1]+\mbox{Var}[X_2]. \]

  • Definition of covariance and correlation By definition, the \(\mbox{Cov}[X_1,X_2]\) is given by \[ \mbox{Cov}[X_1,X_2] = \mbox{E}[(X_1-\mbox{E}[X_1])(X_1-\mbox{E}[X_1])] = \mbox{E}[X_1 X_2]-\mbox{E}[X_1] \mbox{E}[X_2] \] and \[ \mbox{Cov}[X_1,X_2] = \frac{\mbox{Cov}[X_1,X_2]}{\sqrt{\mbox{Var}[X_1]\mbox{Var}[X_2]}}. \] In the case when \(X_1\) and \(X_2\) are independent random variables then \(\mbox{E}[X_1 X_2]=\mbox{E}[X_1] \mbox{E}[X_2]\) and so \(\mbox{Cov}[X_1,X_2]=0\) and hence \(\mbox{Cov}[X_1,X_2]=0\) also. This gives justification of the variance formula for the sum of independent random variables.

  • Joint distribution for independent events. Suppose that \(X_1\) and \(X_2\) are a pair of independent on identically distributed random variables then their joint probability function is given by \[ f_{X_1,X_2}(x_1,x_2) = f_{X_1}(x_1) f_{X_2}(x_2). \]

11.3 Standard distributions

Here, \(f(x;\theta) \equiv f(x| \theta)\) is written to denote a probability function if \(X\) is discrete, or a probability density function if \(X\) is continuous, where \(\theta\) is the parameter of the distribution.

  • Bernoulli, with parameter \(\theta\) (\(0 < \theta < 1\)). A discrete random variable \(X\) with probability function \[ f(x;\theta) ~~=~~ \theta^x (1 - \theta)^{1-x} \quad \qquad x = 0, 1 \]

    \(\mbox{E}[X] ~=~ \theta~\) and \(\mbox{Var}[X] = \theta (1 - \theta)\).

  • Geometric, with parameter \(\theta\) (\(0 < \theta < 1\)). A discrete random variable $X$ with probability function \[ f(x;\theta) = \theta (1 - \theta)^{x-1} \quad \qquad x = 1, 2, \dots \]

    \(\mbox{E}[X] = 1/\theta\) and \(\mbox{Var}[X] = (1 - \theta)/\theta^2\).

  • Binomial, with parameters \(n\) and \(\theta\) (where \(n\) is a known positive integer and \(0 < \theta < 1\)). A discrete random variable \(X\) with probability function \[ f(x;n,\theta) = {n \choose x} \theta^x (1 - \theta)^{n-x} \quad \qquad x = 0, 1, \dots, n \]

    \(\mbox{E}[X] = n \theta\) and \(\mbox{Var}[X] = n \theta(1 - \theta)\).

  • Poisson, with parameter \(\theta\) (\(\theta >0\)). A discrete random variable \(X\) with probability function \[ f(x;\theta) ~~=~~ \frac{\theta^x e^{- \theta}}{x!} \quad \qquad x = 0, 1, \dots \]

    \(\mbox{E}[X] ~=~ \theta~\) and \(~\mbox{Var}[X] ~=~ \theta\).

  • Negative Binomial, with parameters \(r\) and \(\theta\) (\(r > 0\) and \(0 < \theta < 1\)). A discrete random variable \(X\) with probability function \[ f(x;r,~\theta) ~~=~~ {{x-1} \choose {r-1}} \theta^r (1 - \theta)^{x-r} \quad \qquad x = r, r+1, \dots \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{r}{\theta}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{r(1 - \theta)}{\theta^2}\).

  • Uniform, with parameter \(\theta\) \((\theta > 0)\). A continuous random variable \(X\), with probability density function \[ f(x;\theta) ~~=~~ \frac{1}{\theta} \quad \qquad 0 < x < \theta \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{\theta}{2}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{\theta^2}{12}\).

  • Exponential, with parameter \(\lambda\) \((\lambda > 0)\).

    A continuous random variable \(X\), with probability density function \[ f(x;\lambda) ~~=~~ \lambda e^{- \lambda x} \quad \qquad x > 0 \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{1}{\lambda}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{1}{\lambda^2}\).

  • Gamma with parameters \(\alpha\) and \(\beta\) \((\alpha, \beta > 0)\). A continuous random variable \(X\), with probability density function \[ f(x;\alpha, \beta) ~~=~~ \frac{\beta^\alpha x^{\alpha-1} e^{-\beta x}}{\Gamma(\alpha)} \quad \qquad x > 0 \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{\alpha}{\beta}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{\alpha}{\beta^2}\).

    Note: \(\Gamma(\alpha+1) ~=~ \alpha \Gamma(\alpha) ~~ \forall \alpha\) and \(\Gamma(\alpha+1) ~=~ \alpha!\) for integers \(\alpha>1\).

    BEWARE some authors replace \(\beta\) by \(1/\beta\) but still calling it Gamma\((\alpha, \beta)\). You always need to be clear which parameterization is being used.

  • Beta with parameters \(\alpha\) and \(\beta\) \((\alpha, \beta > 0)\). A continuous random variable \(X\), with probability density function \[ f(x;\alpha, \beta) ~~=~~ \frac{x^{\alpha-1} (1 - x)^{\beta-1}}{B(\alpha, \beta)} \quad \qquad 0< x < 1 \] where \[ B(\alpha, \beta) ~~=~~ \int_0^1 x^{\alpha-1} (1-x)^{\beta-1} dx ~~=~~ \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)} \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{\alpha}{\alpha+\beta}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{\alpha \beta}{(\alpha+\beta)^2 (\alpha+\beta+1)}\).

    Also referred to as the Beta\((\alpha, \beta)\) distribution.

  • Normal with parameters \(\mu\) and \(\sigma^2\) \((-\infty < \mu < \infty\) and \(\sigma^2 > 0)\). A continuous random variable \(X\), with probability density function \[ f(x;\mu, \sigma^2) ~~=~~ \frac{1}{\sqrt{2\pi}~\sigma}~\exp \left\{ -\, \frac{1}{2 \sigma^2} (x - \mu)^2 \right\} \quad \qquad -\infty < x < \infty \] \(\mbox{E}[X] ~=~ \mu~\) and \(~\mbox{Var}[X] ~=~ \sigma^2\).

    Also referred to as the N\((\mu, \sigma^2)\) distribution.

  • Pareto with parameters \(\theta\) and \(\alpha\) \((\theta, \alpha > 0)\). A continuous random variable \(X\), with probability density function \[ f(x;\theta, \alpha) ~~=~~ \frac{\alpha \theta^\alpha}{x^{\alpha+1}} \quad \qquad x > \theta \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{\alpha \theta}{(\alpha-1)}~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{\alpha \theta^2}{(\alpha-1)^2 (\alpha-2)} ~~~ (\alpha>2)\).

  • Chi-Square with parameter \(n\) (\(n\) is a positive integer). A continuous random variable \(X\), with probability density function \[ f(x;n) ~~=~~ \left( \frac{1}{2} \right)^{\frac{n}{2}} \frac{x^{\frac{n}{2}-1} e^{-\frac{x}{2}}}{\Gamma(\frac{n}{2})} \quad \qquad x > 0 \]

    \(\mbox{E}[X] ~=~ n~\) and \(~\mbox{Var}[X] ~=~ 2n\).

    Also referred to as the \(\chi^2_n\) distribution, with \(n\) degrees of freedom.

    Note: A \(\chi^2_n\) distribution is also a Gamma\((\frac{n}{2},\frac{1}{2})\) distribution.

  • Student’s t with parameter \(n\) (\(n\) is a positive integer). A continuous random variable \(X\), with probability density function \[ f(x;n) ~~=~~ \frac{\Gamma(\frac{n+1}{2})}{\sqrt{n \pi} ~~ \Gamma(\frac{n}{2}) \left[1+\frac{x^2}{n}\right]^{\frac{n+1}{2}}} \quad \qquad -\infty < x < \infty \]

    \(\mbox{E}[X] ~=~ 0 ~~ (n>1)~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{n}{n-2} ~~ (n>2)\).

    Also referred to as the \(t\) distribution, with \(n\) degrees of freedom.

  • F with parameters \(m\) and \(n\) (\(m,n\) are positive integers). A continuous random variable \(X\), with probability density function \[ f(x;m,n) ~~=~~ \left( \frac{m}{n} \right)^{\frac{m}{2}} \frac{\Gamma(\frac{m+n}{2}) ~ x^{\frac{m}{2}-1}}{\Gamma(\frac{m}{2})\Gamma(\frac{n}{2}) ~ \left[1+\frac{mx}{n}\right]^{\frac{m+n}{2}}} \quad \qquad x > 0 \]

    \(\displaystyle \mbox{E}[X] ~=~ \frac{n}{n-2} ~~ (n>2)~\) and \(~\displaystyle \mbox{Var}[X] ~=~ \frac{2n^2 (m+n-2)}{m(n-2)^2 (n-4)} ~~ (n>4).\)

    Also referred to as the \(F\) distribution, with \(m\) and \(n\) degrees of freedom.

    Note: The order of \(m\) and \(n\) is important.
